# 购物小程序Demo

## 📋介绍

- 一个非常简单的微信购物小程序，用来练手

- 后端使用json-server
- 前端使用小程序原生的组件库和[WeUI组件库](https://wechat-miniprogram.github.io/weui/docs/#weui%E7%BB%84%E4%BB%B6%E5%BA%93%E7%AE%80%E4%BB%8B)----文档写得不清晰，内容少还没例子



## 💻功能

- [x] 展示商品列表
- [x] 商品详情
- [x] 展示评论
- [x] 商品分类
- [x] 搜索商品
- [x] 加入购物车
- [x] 微信授权
- [x] 绑定手机号
- [x] 更换头像



## ⚡️运行项目

先启动json-server服务器

在cmd中打开db.json所在的文件夹

```
json-server --watch db.json --port 5000
```
